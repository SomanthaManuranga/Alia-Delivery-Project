# Care robot

This directory contains simulation files for a scenario in home care. A mobile robot must be able to transport a good to a robot arm at bed side.

This simulation serves for education purpose and has no real application yet.

## Get started

1. Install python 3 or higher
2. Install webots
3. Go to directory /src/3d-sim/worlds and open care-robot.wbt
4. Start simulation (play button at the top)
5. Press 3 then 4 then r. The robot should start to explore the room now.
